## Описание
Приложение исследует влияние давления газа и температуры спекания на плотность твёрдого сплава.
Использует базу данных SQLite, графический интерфейс на Tkinter, построение графиков через Matplotlib.

## Требования
- Python 3.8 или новее
- Библиотеки: numpy, pandas, matplotlib, openpyxl (для экспорта Excel)
- sqlite3 встроена в Python, дополнительная установка не нужна

## Установка зависимостей
Выполните команду:
pip install numpy pandas matplotlib openpyxl

## Запуск
Из командной строки в каталоге с файлом выполните:

python3 project.py

(либо `python project.py` в зависимости от версии Python и ОС)

## Использование
- При запуске откроется окно входа.
- Используйте логин/пароль:
  - Исследователь: login=`researcher`, pass=`pass123`
  - Администратор: login=`admin`, pass=`admin123`
- Исследователь может проводить эксперимент, выбирая материал и параметры.
- Администратор может редактировать коэффициенты и управлять пользователями.
- Результаты можно сохранить в Excel.

## Файлы
- `project.py` — основной файл приложения
- `ceramics.db` — файл базы данных (создаётся автоматически)
